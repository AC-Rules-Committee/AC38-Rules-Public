# AC38 Rule Documents — Public Releases

This repository contains the official published release artifacts for the AC38 Rule Documents.

## Available Downloads

Each release contains:

| File | Description |
|------|-------------|
| `TRv*.pdf` | Technical Regulations |
| `CR75v*.pdf` | AC75 Class Rule |
| `CR40v*.pdf` | AC40 Class Rule |
| `SP75v*.pdf` | AC75 Specification |
| `DC40v*.pdf` | AC40 Documents |
| `AC75Specification_v*.zip` | AC75 Specification supporting files |
| `AC40Documents_v*.zip` | AC40 Documents supporting files |
| `checksums.txt` | SHA-256 checksums for all files |

## Finding Releases

All releases are listed on the [Releases page](../../releases).
